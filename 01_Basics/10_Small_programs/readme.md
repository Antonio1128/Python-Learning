# Small Programs

# Description

This chapter contains small practical programs that combine core Python concepts such as loops, functions, data structures, modules, and file handling.

##  Programs

* **file.txt**
  Sample file used for testing file reading and writing.

* **list_manager**
  Manages a list of numbers (add, display, basic operations).

* **mini_data_analyzer**
  Reads numbers from a file and calculates sum, average, min, and max.

* **number_analyzer**
  Analyzes a number (positive/negative, even/odd).

* **numbers.txt**
  Input file containing numbers for analysis.

* **password_generator**
  Generates random passwords using letters and digits.

* **prime_number.py**
  Checks whether a number is prime.

* **small_calculator**
  Performs basic arithmetic operations (+, -, *, /).

* **student_dictionary**
  Manages students and grades using a dictionary with menu options.

* **task.txt / tasks.txt**
  Files used for storing tasks.

* **to_do_list**
  Simple task manager that saves and loads tasks from a file.

* **word_counter**
  Counts words and characters from a text file.

## 🔹 Concepts Covered

* User input and output
* Conditional statements
* Loops
* Functions
* Lists, sets, dictionaries
* File handling (read/write/append)
* Using modules (random, string)
