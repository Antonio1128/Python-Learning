# Libraries & Modules

This chapter introduces Python modules and libraries, showing how to reuse existing code, organize programs, and extend functionality using built-in and external tools.

## Concepts Covered

* Importing modules (import)

* Importing specific functions (from ... import)

* Using aliases (as)

* Creating custom modules

* Built-in libraries (math, random, datetime)

* External libraries and installation (pip)

* Code organization and reuse
 
### Built-in Libraries (come with Python)

* math → mathematical functions

* random → random values

* datetime → dates and time

* os → interacting with the operating system

* sys → system-related info

* string → text utilities
