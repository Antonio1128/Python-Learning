# Python OOP

Basic Object-Oriented Programming concepts in Python.


## Programs

* class
  Introduction to classes and objects (attributes and methods)

* encapsulation
  Uses private variables and controlled access (e.g. bank account)

* inheritance/
  Base class with child classes that extend or override behavior

* polymorphism
  Different classes using the same method name with different behavior

* magicmethods/
  Special methods like __str__, __len__, __eq__
